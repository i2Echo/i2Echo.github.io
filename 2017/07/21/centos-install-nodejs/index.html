前言：

本文主要记录下在centos6上安装nodejs并实现node版本管理的过程。

一、使用源码安装

准备

yum -y install gcc make gcc-c++ openssl-devel wget

下载源码并解压

wget http://nodejs.org/dist/v6.4.4/node-v6.4.4.tar.gz
# 此处版本号可以更换你想安装的
tar -zvxf node-v6.4.4.tar.gz

编译&安装

# 注意使用node权限
make && make install

验证安装成功

node -v
# 显示版本号即安装成功

二、使用yum安装

安装nodejs

yum install -y nodejs

查看版本

node -v

安装node版本管理工具n

npm install -g n

升级node为最新稳定版

n stable

查看是否升级成功

node -v
# 显示最新版本号v8.2.1

这样就OK了，可以愉快的撸node了
